## ‚ú® Features

- **Modern UI/UX**: Beautiful purple gradient hero section with glassmorphism effects.
- **Dynamic Booking System**: Choose available slots and see them update to "Booked" in real-time.
- **Dark Mode Support**: Seamless transition between light and dark themes using Tailwind's state management.
- **Smooth Animations**: Powered by `Framer Motion` for page transitions and card hover effects.
